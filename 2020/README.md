# Advent of Code - 2020

Solutions to [Advent of Code 2020](https://adventofcode.com/2020) challenges, written in [Java](https://www.oracle.com/java/).

## Running the Solutions

Each solution is contained in a single Java file for simplicity.

### To compile and run:

* `javac Day01.java`
* `java Day01`
